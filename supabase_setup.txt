# Supabase Setup Instructions

Follow these steps to set up your Supabase backend for the Jerseyvault website.

## 1. Create a Supabase Project

Go to https://supabase.com/ and create a new project.

## 2. Get Your API Credentials

In your Supabase project, go to **Settings > API** and find your:
- Project URL
- anon public key

## 3. Update the `supabaseClient.js` File

Open the `src/supabaseClient.js` file in the project and replace the placeholder values with your actual Supabase credentials.

Replace 'YOUR_SUPABASE_URL' with your Project URL.
Replace 'YOUR_SUPABASE_ANON_KEY' with your anon public key.

## 4. Create the `products` Table

In your Supabase project, go to the **SQL Editor** and run the following SQL query to create the `products` table:

```sql
CREATE TABLE products (
  id bigint generated by default as identity primary key,
  name text not null,
  price numeric not null,
  image text
);
```

## 5. Insert Product Data (Optional)

You can insert some sample data into your `products` table by running the following SQL query in the SQL Editor:

```sql
INSERT INTO products (name, price, image)
VALUES
  ('Cyber-Punk Jersey', 79.99, 'https://via.placeholder.com/300x300.png?text=Cyber-Punk+Jersey'),
  ('Galaxy Jersey', 89.99, 'https://via.placeholder.com/300x300.png?text=Galaxy+Jersey'),
  ('Matrix Jersey', 69.99, 'https://via.placeholder.com/300x300.png?text=Matrix+Jersey');
```
