### Prerequisites

- Node.js (version 16 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/zahidhasantonmoy/JerseyVault.git
   ```

2. Navigate to the project directory:
   ```bash
   cd JerseyVault
   ```

3. Install dependencies:

